<h2 align="center">Personal / University Projects:</h2>
<p align="center">
  <a href="https://github.com/stefanbozhilov/energy-forecast">
    <b>Energy Forecast</b> — Daily energy market data pipeline built on Dagster, Delta Lake, and dbt.
    Ingests ENTSO-E market data and Open-Meteo weather signals, transforms them in BigQuery,
    and produces ML-ready feature tables for electricity price forecasting.
  </a>
</p>
<p align="center">
